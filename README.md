## Run Locally

Clone the project

```bash
  git clone https://github.com/MarceeloDominguez/nike-store-app.git
```

Go to the project directory

```bash
  cd nike-store-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
